# ZEN Study コメビュ
元N予備コメビュ。

ZEN Studyの生放送授業において、コメント一覧を表示するChrome拡張です。

サーバーなどへ余計なアクセスはせず、描画エリアに表示されたコメントだけが一覧に表示される仕様です。

- 表示位置の切り替え機能を追加(v2.1)
